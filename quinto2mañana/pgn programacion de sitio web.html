PROGRAMACION DE SITIOS WEB

¿Que es la programacion de sitio web?

La programacion de sitio web:

es hacer una pagina que funcione no solo se trata de como se ve, sino de que los botones hagan algo,que se pueda llenar un formulario, entrar con una cuenta o ver informacion. se usa codigo para que todo eso funcione bien, tanto lo que ves como lo que pasa por detras,es decirle a la computadora que hacer, usando un lenguaje que ella entiende. es como escribir instrucciones para que haga cosas, como mostrar algo, hacer cuentas o hacer que funcione una pagina.

Ejemplo:

  • Que tenga texto,imagen o video.
  • Que tenga botones que se puedan usar
  • Que se pueda completar un formulario,como para registrarse o hacer una compra

    ¿Que se usa para programar sitios web?

    Se utilizan lengujes de progracion y tecnologias web,por Ejemplo:

  • HTML:Estructura basica del sitio (como los ladrillo de una casa :v)
  • CSS:Apariencia (Colores como la pintura,estilos,animacion,formulario.)
  • JAVASCRIPT:Interactividad (Botones,animacion,formulario.)

    • Caracteristica:

  • Interactividad:Permite que el usario haga cosas en la pagina,como apretar botones, mandar mensajes o llenar formulario.
  • Funcionalidad:Hace que la pagina no sea solo un dibujo. gracias a la programacion, las cosas funcionan (Como buscar productos o iniciar sesion.)
  • Organizacion del contenido:Ayuda a mostrar bien la informacion, como dividir una pagina en secciones o mostrar lo que cada persona nesesita ver.
  • Conexion con bases de datos:Puede guardar y mostrar informacion, como usarios registrados, productos o comentarios.
  • Actualizacion dinamica:La pagina puede cambiar sin tener que cargar todo de nuevo, como cuando te llegan notificaciones o se actualiza algo en tiempo real.
  • Adaptabilidad:Permite que el sitio se vea bien tanto en compu com en celular (esto se logra tambien con diseño, pero la programacion ayuda a hacelo posible.)
